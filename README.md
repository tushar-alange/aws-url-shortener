# 🔗 Serverless URL Shortener with Analytics (AWS Project)

A fully serverless URL shortener system built using AWS services with real-time analytics, click tracking, and CI/CD automation using GitHub Actions.

## 📌 Features

- 🔗 Shorten long URLs into compact links
- 🔁 Redirect users to original URLs
- 📊 Track click counts in real-time
- 📈 Store detailed analytics (IP, user-agent, referrer)
- 🌐 Web dashboard hosted on S3
- ⚡ Fully serverless architecture
- 🔄 CI/CD pipeline using GitHub Actions
